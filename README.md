# Tawjihi AI

Tawjihi AI is a cutting-edge banking and finance platform designed to empower - marketing and sales teams with AI-driven insights and tools. This project focuses on enhancing product recommendations, customer profiling, and feedback analysis to drive better decision-making and improve customer engagement.

## Features

- **Customer Profiling**: Gain deep insights into customer behavior and preferences.
- **Product Recommendations**: Tailored suggestions for customer segments, such as:
  - High Net Worth: 92% conversion for Premium Credit Card.
  - Small Business: 90% conversion for Business Credit Line.
- **AI Feedback Analysis**: Monitor AI performance with metrics like:
  - 72% acceptance rate.
  - 90% model accuracy.
- **Dashboard Insights**: Visualize key metrics such as:
  - Customer activity.
  - Conversion rates (e.g., 46% overall).
  - Revenue impact (e.g., 1.2M OMR).
- **System Alerts**: Detect unusual spending patterns and identify marketing opportunities.
- **Follow-Up Scheduling**: Plan and track customer interactions effectively.

## Technology Stack

- **Frontend**: Built with [Next.js](https://nextjs.org/) and [React](https://reactjs.org/).
- **Styling**: Tailwind CSS for responsive and modern UI design.
- **UI Components**: Radix UI and custom-built components for a seamless user experience.
- **State Management**: React Context API and custom hooks.
- **AI Integration**: Advanced AI models for recommendations and feedback analysis.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/amasQIS-ai/Tawjihi_AI.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Tawjihi_AI
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
